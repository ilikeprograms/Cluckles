Download Theme

Once your happy with the changes, click the Download button below and you will get a theme.json file.
You can then use this file to modify the variables in Less using a Less Compiler

To do that, first of all parse the theme.json file (which is included in this page) and then parse the theme.json file using the the modifyVars function.

How you choose to do that isnt part of the BootstrapThemeEditor. However, if you are using Symfony2, you can use BoostrapThemeBundle which is also developed by me.
